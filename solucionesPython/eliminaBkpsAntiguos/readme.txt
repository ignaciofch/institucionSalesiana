Esta carpeta contiene 1 script



Este ejecuta todos los dias a las 8:30h y las 15:30 (elimina_carpetas_backup_antiguas_lunes.py)
eliminando todas las carpetas backup que tengan mas de 7 dias de antiguedad.


Los scripts estan automatizados con crontab
Se pueden revisar o editar con los comandos:
 "crontab -l" para listar tareas
 "crontab -e" para editar
